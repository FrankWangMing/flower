# 示例总结

本文档总结了已创建的 6 个示例，帮助你了解 Flower 的当前能力和使用场景。

## 示例列表

### 1. 基础渲染示例 (`demo.tsx`)
**功能**：最简单的使用场景
- 加载 JSON 格式的几何数据
- 创建 Scene、Cell、Geometry、Material
- 实现渲染循环

**适用场景**：
- 快速原型开发
- 学习基础 API
- 简单 3D 展示

**当前状态**：✅ 完全可用

---

### 2. GLB 模型加载器 (`demo-glb.tsx`)
**功能**：加载 GLB/glTF 2.0 格式的 3D 模型
- 使用 `Loader.GLBLoader()` 加载模型
- 自动提取 vertices 和 indices
- 错误处理和 fallback 机制

**适用场景**：
- 加载从 3D 软件导出的模型
- 产品 3D 展示
- 模型库应用

**当前状态**：✅ 功能完整，需要确保 GLB 文件存在

**待改进**：
- 支持更多格式（OBJ、STL 等）
- 支持材质和纹理加载
- 支持动画数据

---

### 3. 多模型场景 (`demo-multiple.tsx`)
**功能**：在场景中添加多个 3D 对象
- 创建多个 Cell 实例
- 添加到同一个 Scene
- 场景管理能力展示

**适用场景**：
- 复杂场景构建
- 多对象管理
- 场景组合

**当前状态**：✅ 基础功能可用

**待改进**：
- 支持 ModelMatrix 变换（让对象显示在不同位置）
- 对象分组管理
- 场景图（Scene Graph）支持

---

### 4. 交互式相机控制 (`demo-interactive.tsx`)
**功能**：鼠标交互控制相机
- 鼠标拖拽旋转视角
- 实时渲染响应

**适用场景**：
- 3D 查看器
- 交互式展示
- 用户探索场景

**当前状态**：✅ 基础交互可用

**待改进**：
- 鼠标滚轮缩放
- 右键拖拽平移
- 触摸手势支持（移动端）
- 相机动画（平滑过渡）

---

### 5. 材质系统 (`demo-materials.tsx`)
**功能**：创建和使用不同的材质实例
- 创建多个 Material 实例
- 材质管理能力展示

**适用场景**：
- 不同对象使用不同材质
- 材质系统扩展

**当前状态**：🚧 基础结构可用，功能受限

**当前限制**：
- 所有材质都显示红色（shader 硬编码）
- 不支持自定义颜色
- 不支持纹理贴图
- 不支持光照参数

**待改进**：
- 支持自定义颜色（Uniform 传递）
- 支持纹理贴图
- 支持 PBR 材质（粗糙度、金属度等）
- 支持多材质切换

---

### 6. 动画旋转 (`demo-rotation.tsx`)
**功能**：实现简单的旋转动画
- 基于时间的动画
- 更新 ModelMatrix 实现旋转

**适用场景**：
- 动画效果展示
- 动态场景
- 产品展示动画

**当前状态**：🚧 基础实现，需要验证

**待改进**：
- 动画系统抽象（Tween、Timeline）
- 支持多种动画类型（旋转、缩放、平移）
- 关键帧动画
- 骨骼动画支持

---
