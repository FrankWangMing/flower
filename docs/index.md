# Flower - WebGL 3D 渲染引擎

这是一个基于 WebGL 和 React 的 3D 渲染引擎，支持渲染和显示 3D 模型，适用于工业仿真、数据可视化等领域。

## 特性

- 🎨 **灵活的 API** - 提供简洁易用的 API 来创建和管理 3D 场景
- 🚀 **高性能** - 基于原生 WebGL，性能优异
- 📦 **轻量级** - 核心代码精简，易于集成
- 🔧 **TypeScript** - 完整的 TypeScript 类型支持

## 快速开始

### 安装

