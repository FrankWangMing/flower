# 快速启动指南

## 运行示例

### 1. 启动开发服务器

```bash
pnpm doc:dev
```

### 2. 访问文档

打开浏览器访问显示的地址（通常是 `http://localhost:5173`）

### 3. 查看示例

- **首页** (`/`)：基础渲染示例
- **示例页面** (`/guide`)：所有 6 个示例的完整列表

---

## 示例文件列表

所有示例文件位于 `docs/src/` 目录：

1. `demo.tsx` - 基础渲染示例
2. `demo-glb.tsx` - GLB 模型加载器
3. `demo-multiple.tsx` - 多模型场景
4. `demo-interactive.tsx` - 交互式相机控制
5. `demo-materials.tsx` - 材质系统
6. `demo-rotation.tsx` - 动画旋转

---

## 验证方向

运行示例后，请思考以下问题：

### 1. 使用场景
- 你最想用 Flower 做什么？
- 哪个示例最符合你的需求？

### 2. 功能缺失
- 哪些功能是必需的但还没有？
- 哪些功能可以暂时不用？

### 3. 性能表现
- 渲染是否流畅？
- 是否有明显的性能问题？

### 4. API 易用性
- API 是否容易理解？
- 是否有改进空间？

## 问题反馈

如果遇到问题：

1. 检查浏览器控制台是否有错误
2. 确认所有依赖已安装：`pnpm install`
3. 检查文件路径是否正确
